Project

Fly bugs around the tree

Project Overview

Fly Bugs Around the Tree is an exciting website game created during the Global Game Jam 2023, which centered on the theme of "root." In this game, the player controls a Drone bee (represented by a red character) and attempts to catch the elusive Queen bee (a yellow character) without colliding with the other bees (purple characters) swarming around the tree. To make the gameplay more challenging, the Queen bee has been programmed to flee from the Drone bee when they get too close, adding an extra layer of difficulty to the task at hand.

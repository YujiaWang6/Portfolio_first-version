GIF Photo Booth

Project Overview

The online GIF photo booth project showcases a CSS drawing of a Mini Aussie dog figure along with captivating animated backgrounds. There are 5 animated background themes in total. Users can switch between different themes to explore a variety of animated backgrounds by clicking the switcher. Moreover, there are 3 accessories which users can click the green button to add on the dog figure. Users can enjoy the interactive experience by combining different accessories with different animated background.

Recipe Tracker

Project Overview

The Recipe Tracker project enables users to select a dish, manage ingredients, generate a shopping list for missing items, and print the list for convenient shopping. Users click on dish images, mark available ingredients, and generate a shopping list with the "Get Shopping List!" button. They can then print the list and easily shop for the missing ingredients. This project streamlines recipe tracking and ingredient management, enhancing the cooking and shopping experience.

Bead Keychain Design Platform

Project Overview

The Bead Keychain Design Platform is an innovative online solution that replaces traditional bead design boards. It enables users to store their bead collections online, offering convenient CRUD functions for easy management. With the platform's interactive keychain design feature, users can experiment with different bead combinations to create unique keychain designs. The platform also provides a color-based bead grouping feature, allowing users to quickly access beads of the same color. This platform offers designers a virtual space to unleash their creativity and design their perfect keychains.

Fly bugs around the tree

Project Overview

Fly Bugs Around the Tree is an exciting website game created during the Global Game Jam 2023, which centered on the theme of "root." In this game, the player controls a Drone bee (represented by a red character) and attempts to catch the elusive Queen bee (a yellow character) without colliding with the other bees (purple characters) swarming around the tree. To make the gameplay more challenging, the Queen bee has been programmed to flee from the Drone bee when they get too close, adding an extra layer of difficulty to the task at hand.
